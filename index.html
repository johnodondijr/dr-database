Help & Support

Add or edit candidates

Use the Add buttons in Professional or General Jobs. Click any table row or kanban card to edit that candidate.

Attach documents

Click the document icon in a candidate row, paste a Google Drive folder link, then save it.

Track pipeline progress

Use Kanban Board for stage views, Calendar for date-based follow-up, and Reports for summaries.

Export records

Use table exports for CSV downloads. Use Reports export when you need a PDF-style printout.
